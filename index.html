Kites

Place one kite in every row and column. Each kite points into the wind and sends a gust until it hits the edge or another kite. Numbered clouds tell you how many gusts sweep across them.

Fly the kites

Click an empty square to cycle through ↑ → ↓ ← and a small note marker.

Find the hidden wind pattern.

Rules

  1. Place exactly one kite in each row.
  2. Place exactly one kite in each column.
  3. Each kite points up, right, down, or left.
  4. A kite’s gust travels until the board edge or another kite.
  5. Each cloud number tells you exactly how many gusts cross that square.

Why it is new

Kites is not just a placement puzzle. You are deducing both where each kite sits and which way it points, while every choice changes the reach of other gusts because kites block trails.
